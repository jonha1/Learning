Entity Relationship (ER) Diagram is a type of flowchart that illustrates how “entities” 
relate to each other within a system.  ERDs can be used to design databases. PK in an
entity diagram is candidate key that is selected by the database designer to be 
used as an identifying mechanism for the whole entity set.
